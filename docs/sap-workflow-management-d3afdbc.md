# SAP Workflow Management

This page explains how SAP Workflow Management can be integrated with SAP Cloud ALM.

SAP Workflow Management is an automation engine for SAP Cloud ALM. This means you can trigger workflows in SAP Workflow Management from SAP Cloud ALM, for example, as a reaction to an event.

[Setup SAP Workflow Management as Operations Automation engine](https://support.sap.com/en/alm/sap-cloud-alm/operations/expert-portal/operation-automation/sap-workflow-management.html)

Currently, SAP Workflow Management isn't supported for monitoring in SAP Cloud ALM.
