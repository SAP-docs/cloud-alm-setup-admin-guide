# Non-SAP Services and Systems

This page explains how to connect non-SAP services and systems to SAP Cloud ALM to enable monitoring.

Currently, the following monitoring applications in SAP Cloud ALM support non-SAP services and systems:

-   [Integration and Exception Monitoring](https://help.sap.com/docs/cloud-alm/applicationhelp/integration-exception-monitoring)

Follow the instructions for [Adding Non-SAP Services and Systems](https://help.sap.com/docs/cloud-alm/applicationhelp/non-sap-services).
